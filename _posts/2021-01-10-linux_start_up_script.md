# 쉘을 시작할 때 실행(Start-up Script)

### 쉘을 시작할 때 자동적으로 실행되는 명령 지정

1. 별명 지정

    - ```
        alias SHORTCUT='COMMAND'
        ```

         : 커맨드의 별명 지정

        - 예
            - `alias ..='cd ..'`
            - `alias c='clear'`

2. `bashrc` or `zshrc` 수정
