# Machine Learning Modeling
**1) Regression**
  - 1) Learning(Training) -> MSE(Train Data)
  - 2) Validation -> MSE(Test Data)

<br>  

**2) Classification (Binary, Categorical)**
  - 1) Learning -> **CEE**, MSE(가능하긴 함)
  - 2) Validation -> Accuracy, Precision, Recall, F1-Score
